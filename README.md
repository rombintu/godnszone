# DNS Zone tools

### Description 
Alternative Golang module for the [dnszone](https://pypi.org/project/dnszone/) (python-package)

### Dependencies
* Linux*
* bind
* bind9tools

## Install
```bash
$ go get "github.com/rombintu/godnszone"
```

## Usage
```go
package main

import "github.com/rombintu/godnszone"

func main() {
    
}
```
